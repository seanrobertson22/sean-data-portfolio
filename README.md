# COVID-19 Data Analysis (Databricks SQL)

## Overview
This project explores COVID-19 data using **Databricks SQL**. The focus is on querying, transforming, and analyzing public health data to identify trends in cases and deaths over time.

This project is currently focused on **SQL-based analysis only**. Dashboards and visualizations may be added in future iterations.

---

## Tools Used
- Databricks SQL
- SQL (CTEs, aggregations, filtering)

---

## Analysis Highlights
- Daily and cumulative case counts
- Daily and cumulative deaths
- Time-based trend analysis
- Region-level aggregation and comparison

---

## Status
✅ SQL analysis complete  
⏳ Dashboards and visualizations not yet built
